 🌱 Noktat Miah - نقطة مية
📱 فكرة التطبيق
تطبيق ذكي بيساعد المزارعين في حساب كمية المياه المناسبة للري
بناءً على نوع المحصول، نوع التربة، ودرجة الحرارة.
⚙️ المميزات
✔️ حساب كمية المياه تلقائيًا  
✔️ اختيار نوع المحصول  
✔️ تحديد نوع التربة  
✔️ مراعاة درجة الحرارة  
✔️ نصائح زراعية ذكية  
✔️ واجهة بسيطة وسهلة الاستخدام  
🧠 طريقة العمل
يقوم المستخدم بإدخال:
- مساحة الأرض  
- نوع المحصول  
- نوع التربة  
- درجة الحرارة  
ثم يقوم التطبيق بحساب:
👉 كمية المياه المطلوبة  
👉 كمية التوفير في المياه  
🧮 المعادلة المستخدمة
كمية المياه = مساحة الأرض × استهلاك الفدان × معامل التربة × معامل الموسم
